Stand 개인정보처리방침

최종 수정일: 2026년 1월 9일
시행일: 2026년 1월 9일

Move of Today(이하 "회사")는 Stand 애플리케이션(이하 "앱")을 운영하며, 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 회사가 수집하는 정보, 사용 방법, 보호 방법에 대해 설명합니다.

1. 수집하는 개인정보 항목

1.1 필수 수집 정보

항목 수집 목적 보유 기간
건강 및 피트니스 정보 (걸음 수) 일일 목표 달성 여부 확인, 앱 차단 기능 제공 회원 탈퇴 시 즉시 삭제
앱 사용 정보 (설치된 앱 목록, 앱 사용 시간) 차단할 앱 선택, 사용 통계 제공 회원 탈퇴 시 즉시 삭제
기기 정보 (OS 버전, 기기 모델) 서비스 최적화, 오류 분석 회원 탈퇴 시 즉시 삭제

1.2 선택 수집 정보

항목 수집 목적 보유 기간
이메일 주소 계정 생성, 로그인, 고객 지원 회원 탈퇴 시 즉시 삭제
결제 정보 인앱 구매 처리 (Google Play를 통해 처리) 전자상거래법에 따라 5년 보관
⚠️ 중요: 걸음 수 데이터는 기기 내에서만 처리되며, 서버로 전송되지 않습니다. Health Connect를 통해 수집된 데이터는 앱 내에서만 사용됩니다.

2. 개인정보의 수집 방법

  • Health Connect 연동: 사용자가 Health Connect 연동을 허용한 경우 걸음 수 데이터 수집
  • 접근성 서비스: 사용자가 접근성 서비스를 활성화한 경우 앱 사용 정보 수집
  • Firebase Authentication: 사용자가 이메일 로그인을 선택한 경우
  • 사용 데이터 접근 권한: 사용자가 권한을 허용한 경우 앱 사용 통계 수집

3. 개인정보의 이용 목적

3.1 서비스 제공

  • 일일 걸음 목표 달성 여부 확인
  • 목표 미달성 시 선택한 앱 자동 차단
  • 걸음 수 통계 및 성취도 표시
  • 보증금 시스템 운영

3.2 서비스 개선

  • 사용자 행동 분석 및 서비스 최적화
  • 버그 수정 및 안정성 개선
  • 신규 기능 개발

3.3 고객 지원

  • 문의사항 응대
  • 공지사항 전달

4. 개인정보의 공유 및 제공

회사는 사용자의 개인정보를 외부에 제공하지 않습니다. 단, 다음의 경우는 예외로 합니다:

4.1 제3자 서비스 제공업체

제공받는 자 제공 목적 제공 항목
Google LLC (Firebase) 계정 인증, 데이터 저장 이메일 주소, 사용자 ID
Google LLC (Play Billing) 인앱 구매 처리 결제 정보

4.2 법적 요구

  • 법률에 의해 요구되는 경우
  • 수사 목적으로 법령에 따라 수사기관의 요구가 있는 경우
데이터 저장 위치: 사용자 데이터는 기기 내부 저장소에 저장되며, 일부 데이터는 Google Firebase 서버(미국)에 저장될 수 있습니다.

5. 개인정보의 보유 및 이용 기간

  • 회원 정보: 회원 탈퇴 시 즉시 삭제
  • 결제 정보: 전자상거래법에 따라 5년간 보관
  • 서비스 이용 기록: 통신비밀보호법에 따라 3개월 보관

단, 다음의 정보는 명시한 기간 동안 보관합니다:

  • 부정 이용 기록: 1년 (부정 이용 방지 목적)
  • 고객 문의 기록: 3년 (소비자보호법)

6. 개인정보 파기 절차 및 방법

6.1 파기 절차

이용자가 회원 탈퇴를 요청하거나 개인정보의 보유 기간이 경과한 경우, 지체 없이 해당 개인정보를 파기합니다.

6.2 파기 방법

  • 전자적 파일: 복구 불가능한 방법으로 영구 삭제
  • 종이 문서: 분쇄 또는 소각

7. 이용자 및 법정대리인의 권리

이용자는 다음과 같은 권리를 행사할 수 있습니다:

  • 개인정보 열람 요구
  • 개인정보 정정 요구
  • 개인정보 삭제 요구
  • 개인정보 처리 정지 요구
  • 회원 탈퇴 (앱 내 설정 → 구독 취소)

권리 행사는 앱 내 설정 메뉴 또는 아래 연락처를 통해 할 수 있습니다.

8. 개인정보 보호를 위한 기술적/관리적 대책

8.1 기술적 대책

  • 데이터 암호화 (AES-256)
  • Firebase 보안 규칙 적용
  • HTTPS 통신 사용
  • 정기적인 보안 업데이트

8.2 관리적 대책

  • 개인정보 접근 권한 최소화
  • 개인정보 처리 직원 교육
  • 보안 사고 대응 체계 구축

9. 접근성 서비스 사용에 대한 설명

⚠️ 중요: Stand 앱은 다음 목적으로만 접근성 서비스를 사용합니다:
  • 앱 차단 기능: 사용자가 선택한 앱의 실행을 감지하고 목표 미달성 시 차단 화면 표시
  • 앱 사용 시간 추적: 차단 대상 앱의 사용 통계 제공

수집하지 않는 정보:

  • 사용자가 입력하는 텍스트 (비밀번호, 메시지 등)
  • 화면 내용 (스크린샷)
  • 위치 정보
  • 연락처, 통화 기록

접근성 서비스를 통해 수집된 데이터는 기기 내에서만 처리되며 외부로 전송되지 않습니다.

10. 사용 데이터 접근 권한

Stand 앱은 사용 데이터 접근 권한을 다음 목적으로만 사용합니다:

  • 가장 많이 사용하는 앱 추천 (차단 앱 선택 시)
  • 앱 사용 시간 통계 제공

수집된 사용 데이터는 기기 내에서만 처리되며 외부로 전송되지 않습니다.

11. 쿠키 및 유사 기술

Stand 앱은 쿠키를 사용하지 않습니다. 단, Firebase를 통해 다음 정보가 자동으로 수집될 수 있습니다:

  • 기기 식별자 (익명화)
  • 앱 사용 통계 (익명화)
  • 오류 보고서

13. 아동의 개인정보 보호

Stand 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 고의로 아동의 개인정보를 수집하지 않습니다. 만약 아동의 개인정보가 수집되었음을 인지한 경우 즉시 삭제합니다.

14. 개인정보처리방침의 변경

본 개인정보처리방침은 법령, 정책 또는 보안기술의 변경에 따라 내용이 추가, 삭제 및 수정될 수 있습니다. 변경 사항은 시행 7일 전에 앱 내 공지사항 또는 이메일을 통해 공지합니다.

중요한 변경사항의 경우 시행 30일 전에 공지합니다.

15. 고지의 의무

현 개인정보처리방침의 내용 추가, 삭제 및 수정이 있을 시에는 시행일로부터 최소 7일 전에 앱 내 '공지사항'을 통해 고지할 것입니다.
